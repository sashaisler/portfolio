Tools & Technologies

Description

For my final project in my Digital Systems Lab at Columbia, I worked with my two lab partners to create a piano using an FPGA. This project was my first experience using an FPGA and the language VHDL. Working with this new technology and developing with an unfamiliar language and IDE was very challenging, but I learned to celebrate even the smallest successes along the way had a blast with this incredible learning opportunity.
